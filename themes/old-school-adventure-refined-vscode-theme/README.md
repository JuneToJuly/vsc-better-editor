# Old School Adventure Refined

A more balanced version of Old School Adventure.

## Improvements

- less muddy editor background
- brighter parchment text
- stronger contrast between code categories
- richer crimson title bar and active tab
- cleaner moss-green sidebar and status bar
- brighter yellow highlights
- more stone-like neutral surfaces
- more readable comments and line numbers

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\old-school-adventure-refined-theme-1.1.0`
- macOS/Linux: `~/.vscode/extensions/old-school-adventure-refined-theme-1.1.0`

Restart VS Code, run **Preferences: Color Theme**, and select
**Old School Adventure Refined**.
