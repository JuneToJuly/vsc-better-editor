# Catherby Sky

A dark VS Code theme inspired by a bright coastal sky:

- deep navy editor surfaces
- saturated ocean-blue navigation
- pale cloud-white text
- cyan highlights
- soft gold accents drawn from the sunlit platform

## Install locally

1. Copy this folder into your VS Code extensions directory:
   - Windows: `%USERPROFILE%\.vscode\extensions\catherby-sky-theme-1.0.0`
   - macOS/Linux: `~/.vscode/extensions/catherby-sky-theme-1.0.0`
2. Restart VS Code.
3. Run **Preferences: Color Theme**.
4. Select **Catherby Sky**.

You can also package it with `vsce package` after installing `@vscode/vsce`.
