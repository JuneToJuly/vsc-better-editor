# Ocean Sunrise

A bright sunrise theme based on a calm ocean horizon.

Palette:
- clear blue morning sky
- white clouds
- pale gold sunlight
- warm ivory editor
- cool ocean blue
- restrained coral and green syntax accents

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\ocean-sunrise-theme-1.0.0`
- macOS/Linux: `~/.vscode/extensions/ocean-sunrise-theme-1.0.0`

Restart VS Code, run **Preferences: Color Theme**, and select **Ocean Sunrise**.
