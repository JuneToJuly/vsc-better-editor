# Sunlit Crossing Landscape

This iteration gives the interface a vertical landscape composition:

- blue-sky title bar
- misty blue tab strip
- warm sunlight breadcrumb bar
- ivory editor
- yellow-green section headers
- cool railway-shadow bottom panel
- dark forest status bar

## Install

Extract the folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\sunlit-crossing-landscape-theme-1.2.0`
- macOS/Linux: `~/.vscode/extensions/sunlit-crossing-landscape-theme-1.2.0`

Restart VS Code, run **Preferences: Color Theme**, and choose
**Sunlit Crossing Landscape**.
