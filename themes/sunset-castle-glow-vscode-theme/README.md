# Sunset Castle Glow

A refined version of Sunset Castle.

## Changes

- Slightly brighter editor background
- Warmer explorer/sidebar surface
- Lavender title bar
- Warm cream active tab with gold accent
- Purple keywords
- Lavender classes, sky-blue interfaces, and gold enums
- Brighter blue horizon status bar
- Gold used consistently for focus and active highlights

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\sunset-castle-glow-theme-1.1.0`
- macOS/Linux: `~/.vscode/extensions/sunset-castle-glow-theme-1.1.0`

Restart VS Code, run **Preferences: Color Theme**, and select
**Sunset Castle Glow**.
