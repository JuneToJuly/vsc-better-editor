# Sunset Castle

A dark companion to the Sunlit Crossing theme.

Inspired by:
- lavender and rose sunset clouds
- coral horizon light
- warm castle windows
- meadow gold
- deep charcoal twilight
- blue-violet evening sky

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\sunset-castle-theme-1.0.0`
- macOS/Linux: `~/.vscode/extensions/sunset-castle-theme-1.0.0`

Restart VS Code, run **Preferences: Color Theme**, and select **Sunset Castle**.
