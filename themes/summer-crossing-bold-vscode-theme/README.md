# Summer Crossing Bold

A stronger second iteration of the Summer Crossing theme.

Changes from the first version:

- richer olive title and activity bars
- brighter mustard-gold accents
- stronger grass-green strings
- deeper slate-blue functions and types
- improved contrast across tabs, selections, and sidebars
- warm ivory editor background retained

## Install

Copy the extracted folder into your VS Code extensions directory, restart VS Code,
then select **Summer Crossing Bold** from **Preferences: Color Theme**.
