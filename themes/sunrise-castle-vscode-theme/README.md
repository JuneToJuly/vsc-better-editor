# Sunrise Castle

A bright dawn companion to Daylight, Sunset Castle, and Moonlight Castle.

Palette:
- lavender-pink morning sky
- warm peach sunlight
- pale gold highlights
- ivory editor surfaces
- cool blue horizon
- soft green strings
- muted plum keywords

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\sunrise-castle-theme-1.0.0`
- macOS/Linux: `~/.vscode/extensions/sunrise-castle-theme-1.0.0`

Restart VS Code, run **Preferences: Color Theme**, and select **Sunrise Castle**.
