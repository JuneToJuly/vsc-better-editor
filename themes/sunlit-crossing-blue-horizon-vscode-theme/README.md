# Sunlit Crossing: Blue Horizon

This version keeps the painted countryside palette while making the bottom
status bar a clear sky blue rather than blue-green.

## Changes

- Sky-blue status bar: `#8CCBF5`
- Pale blue status-bar hover states
- Rich blue remote indicator
- Darker forest-green activity bar
- Existing blue title bar, misty tabs, warm editor, gold accents, and colorful
  syntax are retained

## Install

Extract the folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\sunlit-crossing-blue-horizon-theme-1.3.0`
- macOS/Linux: `~/.vscode/extensions/sunlit-crossing-blue-horizon-theme-1.3.0`

Restart VS Code, run **Preferences: Color Theme**, and select
**Sunlit Crossing: Blue Horizon**.
