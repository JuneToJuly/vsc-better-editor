# Old School Adventure UI

This iteration moves closer to a classic MMORPG interface rather than only using
its landscape colors.

- weathered stone-gray tabs and controls
- dark brown leather-like borders
- mossy green editor and panels
- parchment-colored text
- crimson banner title bar
- vivid yellow active indicators
- olive and gold status bar

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\old-school-adventure-ui-theme-1.2.0`
- macOS/Linux: `~/.vscode/extensions/old-school-adventure-ui-theme-1.2.0`

Restart VS Code and select **Old School Adventure UI** from
**Preferences: Color Theme**.
