# Moonlight Castle

A cool, dark companion to Sunlit Crossing and Sunset Castle.

Inspired by:
- deep indigo night skies
- silver moonlight
- pale blue clouds
- cool castle stone
- soft window light
- blue-green reflections

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\moonlight-castle-theme-1.0.0`
- macOS/Linux: `~/.vscode/extensions/moonlight-castle-theme-1.0.0`

Restart VS Code, run **Preferences: Color Theme**, and select
**Moonlight Castle**.
