# Catherby Daylight

This revision follows the supplied image much more literally.

The dominant sampled colors are:

- `#0B60C7` — main sky-blue editor
- `#1674D2` — medium-blue raised surfaces
- `#2090E4` — vivid selection and status blue
- `#33A6EF` — bright cyan-blue accents
- `#8CBFEC` — distant cloud blue
- `#BDD4ED` — shaded cloud white
- `#E6EBF4` — bright cloud white
- `#2B447C` — deep station-shadow blue

## Install

Copy the extracted folder to:

- Windows: `%USERPROFILE%\.vscode\extensions\catherby-daylight-theme-1.1.0`
- macOS/Linux: `~/.vscode/extensions/catherby-daylight-theme-1.1.0`

Restart VS Code, open **Preferences: Color Theme**, and select **Catherby Daylight**.
