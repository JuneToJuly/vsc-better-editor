# Sunlit Crossing Sky

This version keeps the warm countryside palette but changes the entire top title region to a clear blue-sky treatment.

## Main change

- Bright sky-blue title bar
- Lighter inactive title bar
- Blue command-center styling
- Pale blue-gray tab strip
- Existing green, gold, ivory, blue, orange, and red syntax palette retained

## Install

Extract the folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\sunlit-crossing-sky-theme-1.1.0`
- macOS/Linux: `~/.vscode/extensions/sunlit-crossing-sky-theme-1.1.0`

Restart VS Code, run **Preferences: Color Theme**, and choose **Sunlit Crossing Sky**.
