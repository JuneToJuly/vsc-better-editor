# Moonlight Castle: Silver Top

This is the original Moonlight Castle theme with only the top chrome changed.

## Changes from the original

- Silver-blue active title bar
- Softer slate inactive title bar
- Brighter silver command center
- Matching silver menu hover treatment

Everything below the title bar keeps the original Moonlight Castle palette.

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\moonlight-castle-silver-top-theme-1.1.1`
- macOS/Linux: `~/.vscode/extensions/moonlight-castle-silver-top-theme-1.1.1`

Restart VS Code, run **Preferences: Color Theme**, and select
**Moonlight Castle: Silver Top**.
