# Old School Adventure

A retro fantasy theme inspired by early-2000s MMORPG visuals.

Palette:
- mossy greens
- worn stone gray
- parchment gold
- deep red banners
- bright yellow highlights
- muted earth tones

## Install

Extract this folder into:

- Windows: `%USERPROFILE%\.vscode\extensions\old-school-adventure-theme-1.0.0`
- macOS/Linux: `~/.vscode/extensions/old-school-adventure-theme-1.0.0`

Restart VS Code, run **Preferences: Color Theme**, and select **Old School Adventure**.
